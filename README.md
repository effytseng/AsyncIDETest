Synchronized Codepan
====================

Quickstart
----------

### Run the Socket.io server

1. Go to /server directory

2. Install necessary module
```
 $ yarn
```

3. Run server
```
 $ yarn dev
```

### Run codepan server

1. Go to /codepan directory

2. Install necessary module
```
 $ yarn
```

3. Run server
```
 $ yarn dev
```
